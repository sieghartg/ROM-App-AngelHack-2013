ROM-App-AngelHack-2013
======================


Hello World!! Welcome!!

Our team created this app in participation with the 2013 AngelHack held in the Davao City.


What it can Do?
===============
This application intends help Humanitarian Organizations monitor their Relief Goods from start to finish while informing the general public of their progress.
This application was concieved with the intention of accomplishing goals set by the organization.
Not only does it can be used in Relief Efforts but it can also be used in any projects with goals intended to be achieved.
